# Cyber Insider - Blog Images Folder

This folder is used to store custom static images for blog posts and reviews.

## Dynamic SVG Feature Images
To make the application lightweight, fast, and completely immune to broken or missing static files, the app now automatically generates **beautiful, high-fidelity, responsive SVGs** directly inside the React components.

If you ever wish to use an external or custom image, simply upload your image files (such as `.jpg` or `.png`) directly into this folder, and reference them in `/src/data/siteData.ts` as `/images/your-image-name.jpg`.
